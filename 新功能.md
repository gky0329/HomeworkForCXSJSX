# AI 提示弹窗（可复用）

本功能在提示用尽后提供 AI 辅助讲解入口，保持原题目内容不变，仅扩展交互能力。

## 功能概述

- 当关卡提示全部显示完后，再点击“提示”按钮会弹出 AI 提示弹窗。
- 弹窗包含两条横栏：
  - 第一栏固定文案：“还有困难？试试问 AI”。
  - 第二栏为输入框，提示语为“输入你的困惑”，右侧有“发送”按钮。
- 点击发送后以 OpenAI 兼容格式向设置页填写的 URL 发送请求，模型与 Key 均来自设置页。
- 使用流式模式显示 AI 输出，发送后显示“请稍后…”，收到内容后开始追加显示。
- 弹窗底部始终有“关闭”按钮。

## 调用方式

在关卡页提示用尽时调用：

1. 组装系统与用户提示，并传入题型参数：
   - `question_type` 由调用方提供（示例：`fill_blank`）。
   - system（填空题）：`请帮助学生学习C++程序设计，给他合适的提示和详细的讲解，但不要直接给出答案。这是一道填空题，题目为{question}`
   - user（填空题）：`我当前在题目中填的空为{answer}（每空之间用@分割），我的困惑为{problem}`
     - `answer` 为当前所有空的作答，用 `@` 分隔。
     - `problem` 为弹窗输入框内容。
   - 其他题型可按同样格式定义对应的 system/user 模板。
2. 通过 `AiHelpDialog` 发送请求并展示结果。

## 代码位置

- 弹窗与流式请求封装：`app/ui/widgets/ai_help_dialog.py`
- 调用入口：`app/ui/pages/challenge_page.py`
- 设置来源：`app/services/app_settings.py` 与 `app/ui/pages/settings_page.py`

## 日志规则

- 发送请求时输出 INFO 日志。
- 收到 200 响应时输出 INFO 日志。
- 429 或 5xx 响应时输出 WARNING 日志。

## 依赖说明

使用 Python 标准库 `urllib` 发送请求，无需新增第三方依赖。
